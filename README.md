# Welcome to PokemonAPI

Search for your favorite Pokemons to know their stats and abilities in this web application using the Pokeapi!

### Technologies used

- React.js

### External modules

- axios
- styled-components

## Build node_modules

In the project directory, run

```
npm install
```

or

```
npm i
```

## Run development mode

In the project directory, run

```
npm start
```

## Build app

In the project directory, run

```
npm build
```
